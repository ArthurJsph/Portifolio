# Meu Portfólio Pessoal

Bem-vindo ao meu portfólio pessoal! Este projeto foi desenvolvido para apresentar as minhas habilidades e experiências como desenvolvedor Full Stack. Aqui você encontrará informações sobre mim, minhas competências técnicas, projetos realizados e formas de entrar em contato.

## Tecnologias Utilizadas

- **React**: Utilizado para a construção da interface do usuário, garantindo uma experiência interativa e dinâmica.
- **Vite**: Ferramenta de build que proporciona um desenvolvimento rápido e eficiente com React.
- **CSS3**: Estilização e layout do site, garantindo um design responsivo e moderno.
- **Vercel**: Hospedagem do portfólio para garantir uma performance rápida e estável.

## Seções do Portfólio

- **Home**: Página inicial com uma introdução sobre mim.
- **Sobre**: Informações detalhadas sobre minha jornada como desenvolvedor.
- **Habilidades**: Descrição das minhas principais habilidades técnicas.
- **Projetos**: Exposição dos projetos que desenvolvi, com links para os repositórios e demonstrações.
- **Contato**: Formas de entrar em contato comigo para oportunidades ou parcerias.

## Funcionalidades

- Design responsivo para garantir uma boa experiência em dispositivos móveis e desktops.
- Navegação fácil e intuitiva entre as seções do site.
- Seção de projetos com detalhes técnicos e links para código-fonte e visualização dos projetos ao vivo.
